# wavefunction_tools

Python utilities for validating functions used in the hydrogen wavefunction (Laguerre, Legendre, Spherical Harmonics, Binomial Coefficient, etc.).

### Usage

```bash
# Create and activate a virtual environment
cd python/
python3 -m venv .venv
source .venv/bin/activate

# Install wavefunction_tools (editable) and dependencies
pip install -e ./

# Run tests
pytest ./
```
